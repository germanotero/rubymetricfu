h1. RubyMetric

Forked from SVN world: "http://wiki.hudson-ci.org/display/HUDSON/Ruby+metrics+plugin":http://wiki.hudson-ci.org/display/HUDSON/Ruby+metrics+plugin

A plugin for "Hudson":https://hudson.dev.java.net/ a Extensible continuous integration engine.

h2. Added features

* A graph which reports the average flog method score over build history.

h2. Requirements

To use RubyMetrics you must have the following Hudson plugins installed:

* Ruby plugin: "http://wiki.hudson-ci.org/display/HUDSON/Ruby+Plugin":http://wiki.hudson-ci.org/display/HUDSON/Ruby+Plugin
* Rake plugin: "http://wiki.hudson-ci.org/display/HUDSON/Rake+plugin":http://wiki.hudson-ci.org/display/HUDSON/Rake+plugin

To use the flog graph you must have:

# Flog gem installed 
# A flog rake task available in your project:

Something like:
<pre></code>  desc "Run flog over significant files"
  task :flog do
    sh "find app -name \\*.rb | xargs flog"
  end
</pre></code>
